# arXiv Corpus Snapshot

This folder contains a fetched arXiv snapshot for Graph RAG experimentation.

Contents:

- `metadata/*.json`: structured metadata records (used for graph population).
- `abstracts/*.md`: abstract-level textual documents.
- `pdfs/*.pdf`: original paper PDFs.
- `arxiv_index.json`: fetch manifest and flattened records list.

Collection query:

- `cat:cs.AI OR cat:cs.LG OR cat:cs.CL`

Record count:

- 55 documents

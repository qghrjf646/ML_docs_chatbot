# 09 - Frontend UX and Evaluation Tabs

## UX objectives

- Keep interaction simple for users while exposing trustworthy evidence.
- Make evaluation transparent for iterative system improvement.

## Main UI zones

1. Chat panel (implemented):
   - question input
   - generated answer
   - retrieval mode, model status, latency

2. Matched Documents panel (implemented):
   - supporting sources with arXiv IDs
   - score and categories
   - abstract/pdf links

3. Evaluation tabs (implemented):
   - Overview: interactions and status
   - Retrieval: Recall@K
   - Generation: faithfulness and relevance
   - End-to-End: latency p95 and citation precision

4. Ingestion action (implemented):
   - one-click ingestion trigger for 55 arXiv documents

## Visual direction

- Light-blue, modern technical look.
- Clean cards, clear hierarchy, and responsive layout.
